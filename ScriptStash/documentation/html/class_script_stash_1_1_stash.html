Detailed Description

A class for stashing in-memory scripts text for run-time reuse (load once and use many times). the Stash is also the root point to script line manipulation, tokens/parameters injections and save & reload operations.

Fill the stash with all files that has given pattern, found under one of the given directories list. Duplicate file names will be ignored. In such case, only first file is loaded on APPEND mode, and only last file is loaded on REFRESH mode. Example:

var folders = new List<string>() { "C:\tax_files1", "C:\tax_files2" };

stash.FillStash(folders, "*.T??", Stash.FillStachMode.APPEND);
Parameters

Dictionary<string, string> ScriptStash.Stash.InjectTokens ( Dictionary< string, string >  externalTokens)

Replace script tokens (parameters) from external dictionary of string key-value pairs apply to all scripts. Example:

// sql files format at folder `C:\sql_files`, is :
// ``SELECT [CSV_FIELDS] FROM USERS WHERE[USERS_CONDITION]``
string sql_fields_list = "USER_NAME AS \"Name\", USER_COUNTRY AS \"Country\"";
string sql_condition_str = "USER_GENDER = 'MALE'";
Stash stash = new Stash("C:\sql_files", "*.sql");
Dictionary<string, string> tokensDict = new Dictionary<string, string>();
tokensDict.Add("[CSV_FIELDS]", sql_fields_list);
tokensDict.Add("[USERS_CONDITION]", sql_condition_str);
Dictionary<string, string> queries = stash.InjectTokens(tokensDict);

Returns
